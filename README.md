Phenotyper

Description: 
`Phenotyper` is a program that converts ICD-10 or HPO terms into gene lists for genetic analysis

All files current as of 26 Mar 2026, including,
- ICD_phecode_map.csv: taken from phecode v1.2 provided with the phers package (https://github.com/hugheylab/phers)
- HPO_phecode_map.csv: taken from phecode v1.2 provided with the phers package (https://github.com/hugheylab/phers)
- phenotype_to_genes.txt: HPO/phenotype-to-gene associations provided publically by The Human Phenotype Ontology (HPO) (https://hpo.jax.org/data/annotations).
